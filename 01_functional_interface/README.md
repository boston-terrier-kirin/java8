## Consumer

## BiConsumer

- 引数2つ

# Predicate
filter で使われている。

## Predicate
- 引数あり
- 戻り値はboolean

## BiPredicate
- 引数2つ
- 戻り値はboolean

# Function
## Function
- 引数1つ
- 戻り値1つ

# UnaryOperator
UnaryOperatorはFunctionを継承していて、引数と戻り値の型が同じ場合に使う。

- 引数1つ
- 戻り値1つ

## BiFunction
- 引数2つ
- 戻り値1つ

# Supplier
- 引数なし
- 戻り値1つ
